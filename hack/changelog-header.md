### Linux

```shell
curl -L https://github.com/camunda-community-hub/camunda-cloud-go-client/releases/download/v{{.Version}}/cc-linux-amd64.tar.gz | tar xzv 
sudo mv cdf /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/camunda-community-hub/camunda-cloud-go-client/releases/download/v{{.Version}}/cc-darwin-amd64.tar.gz | tar xzv
sudo mv cdf /usr/local/bin
```
